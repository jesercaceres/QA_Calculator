# Projeto: Calculadora Simples

![Print da IDE](https://i.imgur.com/byAlFUv.png)

## Descrição
Este projeto consiste em uma calculadora simples que realiza operações básicas, como soma, subtração, multiplicação e divisão.

## Pré-requisitos
- Linguagem de Programação: Java
- Bibliotecas: JUnit 5 (para testes unitários)
- IDE: Intelij

## Estrutura do Projeto
O projeto possui uma classe principal, `Calculadora`, que contém os métodos para realizar as operações matemáticas básicas.

### Classes Utilizadas e suas Funções
- `Calculadora`: Contém os métodos para soma, subtração, multiplicação e divisão.
- `CalculadoraJunitTest`: Contém os testes utilizando Junit5 para os métodos soma, subtração, multiplicação e divisão.

## Implantações
As implantações realizadas dentro da atividade incluem:
- Implementação dos métodos da classe `Calculadora`.
- Testes unitários para garantir o funcionamento correto dos métodos.

## Versionamento
- v1.0: Implementação inicial da classe `Calculadora`.
- v1.1: Adição de testes unitários utilizando JUnit 5.
